Thumbnail Generation for Social Media Feeds

This project is intended to generate suitable image preview thumbnails for social media feeds.
This is acheived by cropping a given image to the focal point of the image.
For this first stage of the project, this "focal point" is almost always a face, if
one appears in the image. If no face is found, the focal point is determined to be in the
centre of the image, and a region 25% the size of the original image is chosen.

The cropped images are then scaled to a standard size, exactly 600x335. Based on Twitter's minumum
feed image size.
