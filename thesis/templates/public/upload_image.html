Thumbnail Generation for Social Media Feeds

This project has two main purposes.
The first is to generate suitable image preview thumbnails for social media feeds.
The second is to highlight the outcomes of different approaches to acheiving this first task.
This project utilises two main technologies: classifiers and neural networks.
